---readme---
In questo zip file è contenuto un ESEMPIO di un file html che, tramite IFRAME, incorpora
un form MicrosoftForm con la possibilità di gestire anteprima grafica e testuale.

Va completato per funzionare.
0) producete il vostro form con MicrosoftForm e recuperate l'indirizzo per la raccolta delle risposte
1) rinominare il file template.htm con il nome definitivo (ad es. index.htm, rateme.htm, ecc.)
2) modificare il nuovo file completandolo con tutte le informazioni necessarie: descrizione, URL definitivo sul
   server che ospiterà il servizio, ecc. Non dimenticate di inserire l'indirizzo del form microsoft
3) modificare o sostituire, a piacimento, l'immagine preview-510.jpg. Potete usare anche altre dimensioni di immagine,
previo test, ma ricordate di inserire le dimensioni finali negli opportuni meta tag nel file html
4) generate o sostituite, a piacimento, le icone standard (favicon.ico ecc.). 
Puo' aiutarvi il servizio: https://favicon.io/favicon-converter/ 	
5) mettete tutto insieme nella stessa cartella, e pubblicate il tutto sul server definitivo. Serve un hosting statico (almeno)
con pochissimo spazio disco. L'url definitivo (ad es. https://www.ilmioserverpersonale.org/ilmioform/) deve essere corretto nel file htm che pubblicate!
